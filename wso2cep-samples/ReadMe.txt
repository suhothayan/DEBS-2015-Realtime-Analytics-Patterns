Running Siddhi on WSO2 CEP
==========================

Download CEP 4.0 pack and copy past the directories 0201 - 0209 to 
<WSO2_CEP_HOME>/samples/artifacts/ diractory. 
To start the CEP server with the preferred sample run following from <WSO2_CEP_HOME>

On Linux:   ./wso2cep-samples.sh -sn <sample number>
On Windows: wso2cep-samples.bat -sn <sample number>

E.g

On Linux:   ./wso2cep-samples.sh -sn 0201
On Windows: wso2cep-samples.bat -sn 0201
